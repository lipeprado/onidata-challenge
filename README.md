## React Redux Boilerplate

#### Tools

* React 16.4
* Redux ( set to manage application state )
* Sass || Scss || Css
* Webpack
* Eslint ( Airbnb )
* Modular styles with sass-loader
* Mocha & Chai for test

#### Features

* Login with username and passwor
* JWT token
* Tests with Mocha and Chai

```
- How to Install

* npm run start

```
